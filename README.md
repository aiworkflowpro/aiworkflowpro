AI Workflow Pro publishes practical tutorials, field notes, and build-in-public workflows for solo builders and technical operators who want AI systems that actually run in production.

The focus is not abstract AI commentary. It is the operating layer around modern AI coding tools: how to structure workspaces, connect tools, keep agents grounded, and ship repeatable automation.

---

## What I'm Building

AI Workflow Pro is a public lab for agentic software work:

- Claude Code workflows for real software projects
- MCP setup, integration, debugging, and operations
- AI agent memory, workspace design, and task routing
- Developer automation for solo founders and small technical teams
- Clean public templates and utilities when they are ready to be inspected

## Open-Source Policy

Public repositories under AI Workflow Pro are clean snapshots for open-source tools, examples, templates, and public docs.

Product source code, membership source packs, private experiments, unreleased skills, and internal automation stay private by default. When a project is published here, it is intended to be searchable, reusable, and safe to inspect without exposing private development history.
